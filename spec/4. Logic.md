## Loops

```
{
  1 =:: i
} then {
  i >> print
  i ?= 2 { skip }

  5 ?= i { end }
  1 +> i
} repeat
```

- `repeat` keyword
- conditions inside loop, can put wherever wants, allows to make `while`, `do while`, and other loops
